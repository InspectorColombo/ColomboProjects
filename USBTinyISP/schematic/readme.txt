USB Tiny ISP programmer. Based on Attiny2313 MCU, running on 12 MHz Crystal.
- Logic Level converter present.
- Micro USB socket was routed incorrect, in mirror, so It doesn't work and must be removed from board.
- One of LED was moved on other pin, comparing original schematic.
- USB pull up resistor moved on other pin too.
- Reverse target power protection present.
- Switch for powering target from +5V USB present.
